# 智慧物业管理系统

## 项目概述
- **名称**: 智慧物业管理系统
- **目标**: 提供完整的小区物业管理解决方案，涵盖楼栋、房屋、业主、费用、报修、公告等核心管理模块
- **技术栈**: Hono + TypeScript + Cloudflare Pages + D1 Database + TailwindCSS

## 功能模块

### 已完成功能
- **数据看板**: 综合统计面板，展示楼栋数、业主数、收费率、报修工单等核心指标，含收费趋势图表
- **楼栋管理**: 楼栋信息的增删改查
- **房屋管理**: 房屋信息管理，支持按楼栋/状态筛选
- **业主管理**: 业主信息管理，支持关键字搜索（姓名/手机号）
- **费用管理**: 账单管理（物业费、水费、电费、停车费等），支持在线缴费、按状态/月份筛选
- **报修管理**: 维修工单管理，支持分类、优先级筛选，工单状态流转（待处理→处理中→已完成）
- **公告管理**: 社区公告发布管理，支持置顶、分类筛选，卡片式展示

### 核心特性
- 响应式设计，适配桌面和移动端
- 可折叠侧边栏导航
- 模态框表单，操作便捷
- Toast 消息提示
- 实时时间显示
- Chart.js 收费趋势图表

## API 接口

| 模块 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 健康检查 | GET | `/api/health` | 服务状态检查 |
| 看板 | GET | `/api/dashboard/stats` | 仪表盘统计数据 |
| 楼栋 | GET/POST | `/api/buildings` | 楼栋列表/新增 |
| 楼栋 | GET/PUT/DELETE | `/api/buildings/:id` | 楼栋详情/编辑/删除 |
| 房屋 | GET/POST | `/api/units` | 房屋列表/新增（支持 ?building_id=&status= 筛选） |
| 房屋 | GET/PUT/DELETE | `/api/units/:id` | 房屋详情/编辑/删除 |
| 业主 | GET/POST | `/api/owners` | 业主列表/新增（支持 ?keyword=&status= 筛选） |
| 业主 | GET/PUT/DELETE | `/api/owners/:id` | 业主详情/编辑/删除 |
| 费用类型 | GET | `/api/bills/fee-types` | 费用类型列表 |
| 账单 | GET/POST | `/api/bills` | 账单列表/新增（支持 ?status=&month=&unit_id= 筛选） |
| 账单统计 | GET | `/api/bills/stats` | 收费统计（?month=） |
| 缴费 | POST | `/api/bills/:id/pay` | 账单缴费 |
| 报修 | GET/POST | `/api/repairs` | 报修列表/新增（支持 ?status=&priority= 筛选） |
| 报修统计 | GET | `/api/repairs/stats` | 报修工单统计 |
| 报修 | GET/PUT/DELETE | `/api/repairs/:id` | 报修详情/编辑/删除 |
| 公告 | GET/POST | `/api/notices` | 公告列表/新增（支持 ?status=&category= 筛选） |
| 公告 | GET/PUT/DELETE | `/api/notices/:id` | 公告详情/编辑/删除 |

## 数据模型

- **buildings** - 楼栋表（名称、地址、楼层数、每层户数）
- **units** - 房屋表（楼栋关联、房号、楼层、面积、类型、状态）
- **owners** - 业主表（姓名、手机、身份证、房屋关联、入住日期）
- **fee_types** - 费用类型表（名称、单价、计量单位）
- **bills** - 账单表（房屋、业主、费用类型、金额、月份、缴费状态）
- **repairs** - 报修表（房屋、业主、标题、分类、优先级、状态、负责人）
- **notices** - 公告表（标题、内容、分类、置顶、状态、发布日期）

## 使用指南

1. 打开系统后默认进入 **数据看板**，查看物业整体运营数据
2. 通过左侧导航菜单切换不同管理模块
3. 每个模块页面顶部有筛选器，可快速筛选数据
4. 点击右上角按钮可新增数据记录
5. 表格中每行操作列可编辑或删除记录
6. 费用管理模块支持一键缴费操作
